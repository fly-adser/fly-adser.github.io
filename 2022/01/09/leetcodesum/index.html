队列

deque 是双边队列，具有队列和栈的性质，在 list 的基础上增加了移动、旋转和增删等

queue模块提供多种类型的队列：

队列方式 特点
queue.Queue 先进先出队列
queue.LifoQueue 后进先出队列
queue.PriorityQueue 优先级队列
queue.deque 双线队列

队列的方法：

方法 说明
put 放数据，Queue.put( )默认有block=True和timeout两个参数。当block=True时，写入是阻塞式的，阻塞时间由timeout确定。当队列q被（其他线程）写满后，这段代码就会阻塞，直至其他线程取走数据。Queue.put（）方法加上 block=False 的参数，即可解决这个隐蔽的问题。但要注意，非阻塞方式写队列，当队列满时会抛出 exception
get 取数据(默认阻塞)，Queue.get([block[, timeout]])获取队列，timeout等待时间
empty 如果队列为空，返回True,反之False
qsize 显示队列中真实存在的元素长度
maxsize 最大支持的队列长度,使用时无括号
join 实际上意味着等到队列为空，再执行别的操作
task_done 在完成一项工作之后，Queue.task_done()函数向任务已经完成的队列发送一个信号
full 如果队列满了，返回True,反之False

堆

python中的堆由heapq模块(小根堆)提供：

函数 描述
heappush(heap, x) 将x压入堆中
heappop(heap) 从堆中弹出最小的元素
heapify(heap) 让列表具备堆特征
heapreplace(heap, x) 弹出最小的元素，并将x压入堆中

解题技巧总结

栈与队列

  1. sd09.用两个栈实现队列
  2. sd30.包含min函数的栈
    • 额外使用一个栈，这个栈的栈顶永远保存最小的数
  3. sd59-II.滑动窗口的最大值
    • 双端队列，且该队列内保持递减
  4. sd59-I.队列的最大值
    • 额外使用一个双端队列，该队列内保持单调递减

链表

  1. sd06.从尾到头打印链表
    • 使用list模拟栈，最后返回栈的倒序即可
  2. sd24.翻转链表
    • 使用三个指针，pre、cur、nxt
    • 循环终止条件：cur!=None。因为最后一个节点仍需要翻转
    • 翻转终止条件：cur=None
  3. sd35.复杂链表的复制
    • 复制每个节点，并插入到原节点之后(只用next指针连接)
    • 为复制节点添加random指针
    • 将原节点和复制节点拆分，注意，原节点也要拆分

字符串

  1. sd05.替换空格
  2. sd58-II.左旋转字符串
  3. sd20.表示数值的字符串
    • 有限状态机
  4. sd67.把字符串转换成整数
    • 越界判断处理逻辑

查找算法

  1. sd03.数组中重复的数
    • 原地交换的思路
  2. sd53-I.在排序数组中查找数字
    • 两次二分查找，一次找到数字的右边界，一次找到数字的左边界
  3. sd53-II.0~n-1中缺失的数组
    • 二分查找算法，终止条件需要根据题目要求进行模拟判断
  4. sd04.二维数组中的查找
    • 从右下角开始遍历，如果小于target，则向右移动；否则，向上移动
  5. sd11.旋转数组的最小数字
    • 当无法确定旋转点在左半边还是右半边时，右指针向左移动一位，缩小范围
  6. sd50.第一个出现一次的字符
    • 使用有序字典

搜索与回溯算法

  1. sd32-1.从上到下打印二叉树I
    • 广度优先遍历使用队列
  2. sd32-2.从上到下打印二叉树II
    • 使用两个指针lastNode和nextNode，来判断该层是否遍历完毕
  3. sd32-3.从上到下打印二叉树III

  4. sd26.树的子结构
    • 树的深度优先遍历优先使用递归
  5. sd27.二叉树的镜像
    • 自上而下递归和自下而上递归的区别
  6. sd28.对称的二叉树
    • 自上而下递归
    • 梳理终止条件
  7. sd12.矩阵中的路径
    • 同一单元格内的字母不允许重复使用：前置为""，后置恢复
  8. sd13.机器人的运动范围
    • 额外使用一个二维数组标识该位置是否访问过
  9. sd34.二叉树中和为某一值得路径
    • 自上而下递归，当该条路径不符合要求或已访问过时，需要从访问记录里pop出来
  10. sd36.二叉搜索树与双向链表
    • 定义pre和cur两个指针，用于节点之间的链接
  11. sd54.二叉搜索树的第k大节点
    • 先递归右子树，再递归左子树
  12. sd55-I.二叉树的深度
  13. sd55-II.平衡二叉树
    • 左子树和右子树深度相差不超过1
  14. sd64.求1+2+...+n

  15. sd68-I.二叉搜索树的最近公共祖先
